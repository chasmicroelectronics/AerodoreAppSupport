Aerodore Controller App for iOS

The Aerodore Controller App is an interface to the climate-controlled Aerodore cabinet. It allows the user to see current temperature and humidity levels and enables the user to set a target temperature via the thermostat slider.

Instructions

Ensure the Aerodore unit is in range and powered up. Ensure bluetooth is enabled on the mobile phone and grant access if the app requests it. The app will search for the Aerodore unit and, once connected, will display the current internal temperature. To view the humidity, tap the circular temperature display. To toggle back to temperature, tap the circular humidity display. The thermostat can be accessed by tapping the 'advanced control' button. Use the slider to set a target temperature.
